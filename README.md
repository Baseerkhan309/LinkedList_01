This repository contains basic implementations of **Linked List Data Structure** in C++.

##  Topics Covered
- Singly Linked List
- Node creation
- Insertion operations
  - Insert at beginning
  - Insert at end
  - Insert at position
- Deletion operations
- Traversal (printing list)

---

##  What is a Linked List?

A **Linked List** is a linear data structure where elements (nodes) are connected using pointers.

Each node contains:
- **Data**
- **Pointer to next node**

Unlike arrays, linked lists are **dynamic in size**.
